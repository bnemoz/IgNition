# IgNition
IgNition: ultra-fast nucleotide-level coordinate normalization for immunoglobulin sequences.


IgNition implements a nucleotide-level coordinate framework for immunoglobulin sequences in which key structural landmarks (e.g., CDR boundaries) are fixed across sequences, enabling rapid alignment-free normalization of antibody repertoires. This is especially suited to train AbLLM at the nucleotide level where embedded space is fixed on the length dimension, facilitating downstream use of these embeddings.
